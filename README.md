## How to use Finance Tracker
***
1. Running the code will instantly bring you into the game.
2. Play and either beat the game or quit.
3. Upon leaving the game, you will be asked if you have an account or if you would like to make a new account to save your score.
4. If you have an account and would like to log in, you must give your username and password.
5. If you would like to make a new account then it will have you ask for you username and then make a password.
6. Your account will be saved.
You will need Python to run this successfully.
