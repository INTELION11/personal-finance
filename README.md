***
This is a tool that you can use to track many things pertaining to your finances, including budgeting, income, and savings goals.

## How to use Finance Tracker
***
1. Running the code will instantly bring you to the login screen.
2. To exit, press the exit button.
3. You can press any of the buttons to bring you to the many
4. If you have an account and would like to log in, you must give your username and password.
5. When make a new account, it will have you ask for you username and then make a password.
6. Your account will be saved.
You will need Python to run this successfully.
